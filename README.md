# 🌍 Tourists - Travel Website

This is a simple travel website where users can see places, check tour packages, and use login and register forms.

---

## 🚀 Features

* Home page
* Destinations page
* Tour packages
* About section
* Contact form
* Register and Login pages
* Works on mobile and desktop

---

## 🛠️ Technologies Used

* HTML
* CSS
* Bootstrap
* JavaScript

---

## 📂 Project Files

```
index.html
style.css
images/
```

---

## ⚙️ How to Run

1. Download or clone the project
2. Open `index.html` in your browser
